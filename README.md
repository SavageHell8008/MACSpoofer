### Installation

- Just, Clone this repository 
  ```
  git clone https://github.com/Secure-Scholar/MACSpoofer
  ```
- Now go to the cloned directory 
  ```
  $ cd MACSpoofer
  $ pip3 install -r requirements.txt
  $ python3 MACSpoofer.py
  ```
